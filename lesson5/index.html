GIT flow
GIT clone - для того, чтобы "склонить" репозиторий
GIT pull - для того, чтобы получить изменения с репозитория
GIT checkout -b branchname -создать новою ветку и переключиться на н
GIT strong add. -добавить изменения, которые сделали локально
GIT commit -m "commit text" - подписать их
GITpush origin branchname - запушить на git сервер
